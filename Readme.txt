For run the wourd count and validator python script first install python to your pc.
Here is the details:
https://www.python.org/downloads/

For configure the script you have to write the words wich you want to ignore in the badwords.txt, and then copy your input to the input.txt.
Then run the code for the following command:
python ./wordcount.py yourInputfile
(add your input file in a commandline parameter)

The script created a cleaned source of your input, in that the bad-words will be replaced by *. This file will be saved in 'bbe' file. 

You find some statistics from your source in the out. txt file.
